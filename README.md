# 🏫 School Management Information System (School MIS)

A comprehensive, modern school management system built with Next.js 14, TypeScript, Drizzle ORM, and PostgreSQL. Manage students, teachers, classes, exams, fees, library, attendance, timetable, events, hostel, transport, payroll, and more.

## ✨ Features

### 📚 Core Modules

- **Student Management** - Complete student lifecycle management from admission to graduation
- **Teacher Management** - Faculty directory with departments, subjects, and qualifications
- **Class & Section Management** - Organize academic structure with classes and sections
- **Subject Management** - Curriculum management with mandatory and optional subjects
- **Attendance Tracking** - Daily attendance with multiple status types (present, absent, late, half-day)
- **Examination System** - Schedule exams and manage examination types
- **Results Management** - Record and analyze student performance with automatic grading
- **Fee Management** - Track fees, payments, pending dues, and generate receipts
- **Library Management** - Book inventory, issue/return tracking, and fine management
- **Timetable Management** - Create and manage class schedules and room assignments
- **Event Management** - School events, activities, and calendar management
- **Hostel Management** - Student accommodation and room allocation
- **Transport Management** - Vehicle routes and student transport assignments
- **Payroll Management** - Teacher salary processing and payment tracking

### 🎨 User Experience

- **Modern UI** - Beautiful, responsive design with dark mode support
- **Dashboard Analytics** - Real-time statistics and data visualization
- **Global Search** - Quick search across students, teachers, books, and more
- **Role-Based Access** - Secure access control for different user roles
- **Authentication** - NextAuth.js with credential-based authentication
- **Mobile Responsive** - Fully responsive design works on all devices
- **Notifications** - Real-time notifications for important events

### 🛠 Technical Features

- **TypeScript** - Full type safety across the entire application
- **Server Components** - Next.js 14 App Router with React Server Components
- **Drizzle ORM** - Type-safe database queries with PostgreSQL
- **API Routes** - RESTful API endpoints for all CRUD operations
- **Form Validation** - Client and server-side validation with Zod
- **Data Tables** - Advanced tables with sorting, filtering, and pagination
- **Charts & Graphs** - Data visualization for analytics and reports
- **File Upload** - Profile photo upload support
- **Export/Print** - Print and export functionality for reports
- **Rate Limiting** - API rate limiting for security
- **Error Handling** - Comprehensive error boundaries and error pages
